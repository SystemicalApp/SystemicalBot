## The helper bot for the SystemicalApp.

This bot is planned to be a helper with the SystemicalApp, and to be a pseudo-alternative to the plural kit bot.

(I would add integration to plurakit if it was possible!)


***This is currently very early in development. Expect bugs, missing features, and just generally a somewhat poor experience for a little bit.***

## Features (Early WIP, this will change a lot)


### Create Journal

| Option name          | Required  | Type       | Default Value |  Formatting                    |
| -------------------- | --------- | ---------- | ------------- | ------------------------------ |
| Title                | ✅        | String     | No default    | Not required, accepts markdown |
| Content              | ✅        | String     | No default    | Not required, accepts markdown |
| Date                 | ❌        | String     | Todays date   | MM/DD/YYYY                     |
| thumbnail_attachment | ❌        | Attachment | No default    | Any image file I guess         |
| thumbnail_url        | ❌        | String     | No default    | Any URL to an image            |

Note: The database is not setup right now. This responds currently with what the *view journal* command will do in the future, once the database is setup and I figure out authentication through convex.
